# 格式转换

## 音频转换器

### 使用方法

#### 添加待转换文件

将待转换的文件放到`converter/Audio2WAV/input`文件夹中，然后运行Audio2WAV.py

```shell
cd converter/Audio2WAV
python Audio2WAV.py
```

#### 获取转换成工后的文件

转换后的文件在`converter/Audio2WAV/output`文件夹中
